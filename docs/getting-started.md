# Getting Started

Welcome — let's set up everything you need before lesson 1.

## What you need

You'll use **different tools** for different modules. Don't install everything at once — install each as you reach the relevant module to avoid frustration.

| Module | Tool | Disk space |
|---|---|---|
| 1. Java Foundations | [JDK 21+](https://adoptium.net/) + [VS Code](https://code.visualstudio.com/) with the *Extension Pack for Java* | ~500 MB |
| 2. Kotlin | [IntelliJ IDEA Community](https://www.jetbrains.com/idea/download/) | ~2 GB |
| 3. Android Native | [Android Studio](https://developer.android.com/studio) (latest) | ~10 GB (includes SDK + emulator) |
| 4. Flutter | [Flutter SDK](https://docs.flutter.dev/get-started/install) + Android Studio + VS Code | ~10 GB |

## Hardware

- **8 GB RAM minimum**, 16 GB recommended (Android emulator is RAM-hungry)
- **50 GB free disk** if you'll do everything
- **macOS / Windows / Linux** — all supported
- **Physical Android device** is optional but strongly recommended for Modules 3 & 4

## Recommended order

1. Finish Module 1 (Java) first. It's the longest but gives you the strongest foundation. **Don't skip OOP** — it's the entire mental model used in Android.
2. Move to Module 2 (Kotlin). You'll notice how much cleaner everything feels.
3. Pick Module 3 OR Module 4 — Android Native if you want depth, Flutter if you want speed-to-market.
4. Optionally do the other one.
5. Capstone.

## Time investment

- **Per lesson**: 15–30 min reading + 30–60 min hands-on practice
- **Per module**: 20–40 hours
- **Full course**: 120–200 hours (3–6 months at 5–10 hrs/week)
